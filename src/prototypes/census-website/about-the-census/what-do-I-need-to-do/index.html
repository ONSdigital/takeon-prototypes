If you live in one of the four census rehearsal areas, we’ll send you a letter asking you to take part in the rehearsal. This is a practice run to make sure we get the real thing right.

The rehearsal areas are:

  • Carlisle
  • Ceredigion
  • Hackney
  • Tower Hamlets

The letter we send you will have a unique access code which you can use to complete your census questionnaire online. You can do it in one go or save as you go along to come back to it later.

It’s a lot cheaper if everyone fills the questionnaire in online, instead of being sent a paper copy through the post. However, you can ask for a paper questionnaire to fill in if you need to.

If you do not live in one of the rehearsal areas, you do not need to do anything right now. We’ll be in touch in early 2021 about the census.
